# Windows Access Control & Policy Enforcement Lab

## Objective
This lab focused on configuring and enforcing access control policies in a Windows environment. The goal was to restrict unauthorized access, enforce administrative privileges, and understand how system policies control user permissions and security behavior.

---

### Skills Learned
- User and group permission management
- Administrative policy enforcement
- Access restriction techniques
- Security policy configuration using PowerShell
- Understanding of Windows security controls

---

### Tools Used
- Windows Server / Windows OS
- PowerShell
- Local Security Policy
- Server Manager
- Event Viewer

---

## Steps

*Ref 1: Policy Configuration*  
Configured security policies using local policy settings.

*Ref 2: Administrative Enforcement*  
Applied administrative restrictions to enforce security rules.

*Ref 3: Permission Management*  
Restricted file and system access to admin-only users.

*Ref 4: Server Configuration*  
Used Server Manager to apply and manage system roles and policies.

*Ref 5: Policy Verification*  
Validated enforcement through system behavior and access tests.

---

## Key Outcome
Successfully implemented and enforced access control mechanisms, ensuring only authorized users could access sensitive resources.
